# Sentinel: Linux Security & Hardening Suite 🛡️

Sentinel is a Bash-based automation tool designed to harden Linux environments by automating critical security tasks.

## ✨ Key Features
* **Firewall Configuration:** Automatically sets up UFW rules.
* **SSH Hardening:** Secures the SSH protocol and disables root login.
* **Security Auditing:** Generates real-time health and network listener reports.

## 🚀 How to Run
1. `chmod +x sentinel.sh`
2. `sudo ./sentinel.sh`
